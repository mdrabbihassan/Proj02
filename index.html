Design

Glassmorphism card

Subtle glass surface with floating motion and 3D hover tilt.

Motion

Slow float animation

Each card uses the same keyframes with staggered delays to feel organic.

Interact

Hover pop

Hover to lift and slightly rotate the card for tactile feedback.

Layout

Responsive grid

Grid collapses from 3 → 2 → 1 column depending on viewport width.

Shadows

Depth with shadows

Layered shadows and inner highlights create depth and contrast.

Pure CSS

No JavaScript needed
